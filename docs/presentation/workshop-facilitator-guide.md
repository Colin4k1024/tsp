# Workshop Facilitator Guide

这份讲稿面向内部培训和团队赋能。重点不是单次汇报，而是如何带一场 30 到 90 分钟的培训，让团队真正理解主链、specialist 和 vertical 材料怎么配合。

## 1. 30 分钟版

### 第一段：平台结构

- 讲 8 角色矩阵
- 讲 `/team-*` 主链
- 讲 handoff 为什么是事实收口点

### 第二段：能力升级

- 讲 `/tdd`
- 讲 `/harness-audit`
- 讲 runtime 与显式命令分层

### 第三段：怎么选材料

- 讲 examples 索引
- 讲 route map
- 讲 capability matrix

## 2. 60 分钟版

### 加强内容

- 加一段 vertical 选择演示
- 加一段 Claude / Codex 起手对比
- 加一段 specialist 结果回收到 handoff 的示例

## 3. 90 分钟版

### 加强内容

- 选一个完整 vertical 做 walkthrough
- 选一个非完整听众路线，演示如何从 audience route map 反推讲法
- 补一段 QA / DevOps 如何接 review / release
