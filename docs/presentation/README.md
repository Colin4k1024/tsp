# Presentation Materials

这组材料用于向团队、管理层或新接入项目说明 Team Skills Platform 当前能力，不参与平台生成链路，但属于正式可交付辅助资产。

## 1. 文件说明

- [generate_ppt.py](generate_ppt.py)：生成平台总览 PPTX 的脚本。
- [generate_capability_matrix.py](generate_capability_matrix.py)：生成能力矩阵 PPTX 的脚本。
- [presentation-talk-track.md](presentation-talk-track.md)：5 分钟 / 15 分钟口头讲解提词稿。
- [audience-presentation-route-map.md](audience-presentation-route-map.md)：按听众选择演示路线的导览页。
- [executive-briefing-talk-track.md](executive-briefing-talk-track.md)：面向管理层的讲解稿。
- [implementation-onboarding-brief.md](implementation-onboarding-brief.md)：面向实施接入的讲解稿。
- [workshop-facilitator-guide.md](workshop-facilitator-guide.md)：面向团队培训的讲解稿。
- [vertical-scenario-route-map.md](vertical-scenario-route-map.md)：vertical 场景的模板 / 脚本 / demo 成熟度导览页。
- [../runbooks/vertical-scenario-capability-matrix.md](../runbooks/vertical-scenario-capability-matrix.md)：markdown 版 vertical 场景材料覆盖矩阵。

## 2. 适合什么时候看

- 想快速向团队解释“这不是普通 prompts 仓库，而是角色化协作平台”时。
- 想说明 memory persistence、error experience、parallel execution 这类新增能力时。
- 想做 onboarding、方案汇报或阶段性成果同步时。
- 想先按 9 个完整 vertical 闭环选一条最适合当前听众的演示路线时。
- 想按听众而不是按材料名选择讲法时。
