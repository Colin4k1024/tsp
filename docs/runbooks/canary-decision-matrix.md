# 灰度决策表与症状速查

本文用于灰度过程中快速判断是继续放量、暂停观察还是回滚。

## 1. 决策矩阵

| 现象 | 建议动作 |
|------|----------|
| 指标稳定、无新增错误 | 继续按计划放量 |
| 指标轻微波动，但仍在阈值内 | 暂停放量，继续观察 |
| 错误率明显上升或关键路径失败 | 立即停止放量，评估回滚 |
| 用户反馈与监控同时异常 | 优先回滚或限流 |

## 2. 重点观察

- 错误率
- 接口耗时
- 关键业务成功率
- 前端主路径反馈时延

## 3. 常见症状

- 新错误集中在某个权限或特定端
- 成功率下降但系统未完全不可用
- 页面可打开，但关键动作失败

## 4. 常见错误

- 只看技术指标，不看业务成功率
- 发现波动后继续放量
- 暂停放量后没有明确下一次判断时间点
