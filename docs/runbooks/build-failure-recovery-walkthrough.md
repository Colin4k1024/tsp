# 构建失败处理演练

本文演示构建失败或校验失败时，如何定位、修复、复验并把问题结构化回收到主链。

## 1. 场景

- 构建脚本失败
- 校验脚本失败
- 文档链接或生成产物不一致

## 2. 推荐链路

1. `/team-execute`
2. `/build-fix`
3. `/verify`
4. `/handoff`
5. `/team-review`

## 3. 关键输出

- 根因定位
- 修复动作
- 复验结果
- 是否存在剩余风险

## 4. 常见错误

- 只修表象，不记录根因
- 构建恢复后不补复验
- 没把失败影响范围说明白
